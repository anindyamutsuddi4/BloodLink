- **User Management**: Registration and authentication with role-based access.

### ðŸ”¹ Installation / Running Locally

1. **Clone the repository**  
   `git clone https://github.com/yourusername/BloodLink.git`

2. **Navigate to the project folder**  
   `cd BloodLink`

3. **Install frontend dependencies**  
   `npm install`

4. **Navigate to the backend folder**  
   `cd server`

5. **Install backend dependencies**  
   `npm install`

6. **Create a `.env` file** with necessary Firebase and MongoDB credentials

7. **Run the backend server**  
   `npm start`

8. **Run the frontend**  
   `npm start`

9. **Open the app in your browser**  
  http://localhost:3000
